gRaphaël Bar Chart

gRaphaël is a library built on top of Raphaël. It's not documented at all by the author, although kennyshen has started some documentation. The API is not great, with lots of functions that have several parameters that should take parameter objects instead. I do like Raphaël, and gRaphaël at least shows the promise of the plugin API.
